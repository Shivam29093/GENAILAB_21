
# 🎯 Lab 3: Gaussian Mixture Model (GMM)

## 📌 Objective

To implement **Gaussian Mixture Models (GMM)** for probabilistic clustering and understand how data can be modeled as a mixture of multiple Gaussian distributions.

---

## 🧠 What is GMM?

Gaussian Mixture Model is an unsupervised learning algorithm that assumes data is generated from a combination of several Gaussian distributions with unknown parameters.

---

## 🛠️ Techniques Used

* Gaussian Mixture Models (GMM)
* Expectation-Maximization (EM) Algorithm
* Probabilistic clustering

---

## ⚙️ Workflow

1. Load and preprocess dataset
2. Initialize GMM parameters
3. Apply **Expectation-Maximization (EM)**:

   * E-step: Estimate probabilities
   * M-step: Update parameters
4. Assign clusters based on probabilities
5. Visualize clustering results

---

## 📊 Concepts Covered

* Soft clustering (probabilistic assignment)

* Difference between K-Means and GMM
